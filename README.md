Tento projekt je vypracování na základě zadaného úkolu do hodiny programování na střední škole.

Zadáním úkolu bylo vytvořit aplikaci, která by volala nějaké volně dostupné API a na základě informací obdržených z API vyprodukovala nějaký výstup.

Já jsem se rozhodl udělat generátor obrázků obsahující základní informace a rank mého profilu ve hre Overwatch. Důvodem bylo, abych pak tento obrázek měl ukázaný na ploše a na pozadí by se mi automaticky fetchovali aktuální informace.
